# Day 47 (Dec.23.2021 THU)
- JDBC
- MySQL
    - Installation
    - Schema
    - CREATE SCHEMA `scott` DEFAULT CHARACTER SET utf8mb4 ;
- MySQL Option
    - PK : Primary Key
    - NN : Not Null
    - UQ : Unique Index
    - B 
    - UN : Unsigned : 0과 양수만 허용
    - ZF : Zerofill : 숫자 자리 수가 고정된 경우 빈 자리에 0을 넣음
    - AI : Auto Increment : 숫자일 경우 다음 로우가 저장될 때 자동으로 1 증가
    - Default now() : 날짜 컬럼의 기본값 현재 시간으로
- MySQL vs. OracleDB
    - AUTO_INCREMENT : sequence 대신
    - VARCHAR2 없고 VARCHAR
    - CONSTRAINT 없이 제약 사항 표시
    - 테이블 이름 앞에 반드시 `스키마이름' 붙여서 사용
- MySQL Datatype
    - INT : 정수 자료형
    - VARCHAR : 문자 자료형(가변길이)
    - TEXT : 긴 문자열은 TEXT에 저장
    - DATETIME : 날짜 자료형 저장
    - TINYINT : -128에서 127까지 저장, 1/0만 저장해 불 값 표현

# Day 51 (Dec.29 WEN)

- 쿠키 : 사이트 접속시 개인장치에 다운로드 되고 브라우저에 저장되는 텍스트 파일
    - 사이트가 접속장치 인식 가능
    - 접속자의 설정과 과거 이용내역 일부 데이터 저장
    - 만료일이 있음
    - 브라우저 닫을때 자동으로 삭제되는 쿠키있음
- 쿠키 사용
    - 필수쿠키 : 페이지 탐색, 보안 영역, 검색 등 사이트의 기본기능을 활성화
    - 기능쿠키 : 접속자의 지역 및 언어 등 형태 및 외관에 영향을 줄 수 있는 접속자 설정을 저장하도록 허용
    - 성능쿠키 : 정보의 익명 수집/보고, 상호작용 이해에 도움, 통계자료 제공함으로 사이트 운영자가 더욱 최적화된 웹사이트 개발에 기여

- pageContext, request, session, application : 서버에 존재하며 이용할 수 있는 저장소
