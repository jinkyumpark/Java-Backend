# Day 47 (Dec.23.2021 THU)
- JDBC
- MySQL
    - Installation
    - Schema
    - CREATE SCHEMA `scott` DEFAULT CHARACTER SET utf8mb4 ;
- MySQL Option
    - PK : Primary Key
    - NN : Not Null
    - UQ : Unique Index
    - B 
    - UN : Unsigned : 0과 양수만 허용
    - ZF : Zerofill : 숫자 자리 수가 고정된 경우 빈 자리에 0을 넣음
    - AI : Auto Increment : 숫자일 경우 다음 로우가 저장될 때 자동으로 1 증가
    - Default now() : 날짜 컬럼의 기본값 현재 시간으로
- MySQL vs. OracleDB
    - AUTO_INCREMENT : sequence 대신
    - VARCHAR2 없고 VARCHAR
    - CONSTRAINT 없이 제약 사항 표시
    - 테이블 이름 앞에 반드시 `스키마이름' 붙여서 사용
- MySQL Datatype
    - INT : 정수 자료형
    - VARCHAR : 문자 자료형(가변길이)
    - TEXT : 긴 문자열은 TEXT에 저장
    - DATETIME : 날짜 자료형 저장
    - TINYINT : -128에서 127까지 저장, 1/0만 저장해 불 값 표현
